#about-me

Interest in computers

I started coding in 2018 when I was 14 (more on that later). Before that, I was an amateur computer user, that is, I knew various things, such as Microsoft Office software, Antivirus and Windows installation, Theories and history of computers. The reason I studied them was that I participated in the Science Olympiad in Computer Science. My computer science teacher was a big reason for that. He noticed during class that I was gifted in the Paint program. And I knew Paint from my 1st grade. It was fun for me to draw Marvel heroes on the computer my one and only mother in the house bought for the office.

My first program

Thus, for the 4th year in a row, I became the regional champion in computer science. During this time, programming managed to attract me. The kids learning programming at our school got me excited and I wrote the code for the first time. The name of the programming language was Python. I had tried Pascal and HTML in a simple incomprehensible form before, but in Python I felt like a real programmer. The colors in VScode, the ease of finding bugs, and most importantly its ease of use, completely captivated me. To learn it, I used Eric Metiz's book, Let's Learn Python. My first project was "Tic-Tac-Toe Game". "Excellent, I'm writing code, I know Python perfectly, now?"

New choice, New career

Now the "OMUC(One Million Uzbek Coders)" project has come into my life. There I discovered new words for myself, such as Front-End, Full-Stack, and here I transferred from Python programming to JavaScript programming. Because the demand for Python in our region at that time was very low. In my Web Programming career, which started with small projects, I completed various courses, earned my first income as a freelancer, and more importantly, realized that I knew nothing. I still have a lot to learn. I am currently working on programming in NextJS, ReactJS, HTML, CSS and others...
